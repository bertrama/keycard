# Keycard

Keycard provides authentication support and user/request information, especially
in Rails applications.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'keycard'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install keycard
